# tms-oversikt-mikrofrontend

Mikrofrontend som vises på Min Side.

## Kom i gang

1. Bygg tms-oversikt-mikrofrontend ved å kjøre npm run build
2. Start appen lokalt ved å kjøre npm run dev
3. Appen nås på http://localhost:3000
